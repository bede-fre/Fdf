# FDF


## Download format :

```
git clone https://github.com/bede-fre/fdf_42.git
```

## Compilation :

```
cd fdf/

make

./fdf map color_bottom color_top
```
### Example :
```
./fdf test_maps/42.fdf 0xFF0000 0x00FF00
```

## Commands :

* W     Up
* S     Down
* D     Right
* A     Left

* Q     Left rotation
* E     Right rotation
* X     Enable/disable auto-rotation

* Z     Dezoom
* C     Zoom

* R     Up height
* F     Down height

* SPACE Reset
* ESC   Quit
